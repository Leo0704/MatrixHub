# Claude Code 命令参考

本文档按常用性分类列出 Claude Code CLI 的所有可用命令。

---

## ⭐⭐⭐ 高频常用（日常必备）

这些命令在日常开发中使用频率最高，建议熟练掌握。

| 命令 | 说明 |
|------|------|
| `/help` | 显示所有可用命令和简要说明，忘记命令时随时查看 |
| `/clear` | 清空当前对话，释放上下文空间，开始全新对话（之前的对话内容会被完全清除） |
| `/compact` | 压缩对话历史，保留关键摘要信息，腾出上下文空间但不会丢失重要上下文 |
| `/diff` | 查看代码变更，显示未提交的修改以及每一轮对话后的代码差异 |
| `/plan` | 进入计划模式，让 Claude 先制定实施方案再执行，适合复杂任务 |
| `/context` | 可视化显示当前上下文使用情况，用彩色网格直观展示内存占用 |
| `/cost` | 统计当前会话的 API 调用成本和持续时间，帮你了解花费 |
| `/copy` | 一键复制 Claude 最后的回复内容或指定代码块到剪贴板 |

---

## ⭐⭐ 中等常用（定期使用）

这些命令在特定场景下经常使用，建议了解。

### 代码审查与调试

| 命令 | 说明 |
|------|------|
| `/review` | 对 Pull Request 进行全面代码审查，检查代码质量、潜在问题和改进建议 |
| `/pr-comments` | 获取指定 GitHub PR 的所有评论和讨论内容，快速了解他人反馈 |
| `/security-review` | 对当前分支的代码变更进行安全审查，检查是否存在安全漏洞 |
| `/debug` | 读取会话调试日志，帮助诊断 Claude Code 运行中的问题 |
| `/doctor` | 全面检查 Claude Code 安装状态和配置是否正确，排查环境问题 |

### 会话管理

| 命令 | 说明 |
|------|------|
| `/resume` | 恢复之前中断的对话，继续之前的工作（会话历史会被保留） |
| `/rewind` | 时间回溯，将代码和对话状态恢复到之前的某个节点，撤销操作 |
| `/fork` | 从当前点创建对话分支，可以探索不同方案而不影响主线 |
| `/rename` | 给当前对话起一个有意义的名字，方便后续查找和管理 |
| `/export` | 将整个对话导出为文件或复制到剪贴板，便于分享或存档 |
| `/tasks` | 查看和管理后台运行的任务，如长时间运行的构建或测试 |

### 开发工具

| 命令 | 说明 |
|------|------|
| `/init` | 在项目中初始化 CLAUDE.md 文件，自动分析代码库并生成项目说明文档 |
| `/simplify` | 智能审查代码变更，找出可以简化、复用或优化的地方并自动修复 |
| `/batch` | 大规模批量操作，将一个任务在 5-30 个隔离的工作树中并行执行 |
| `/add-dir` | 添加额外的目录到工作区，让 Claude 可以访问多个项目文件夹 |

## ⭐ 低频使用（偶尔需要）

这些命令在特定情况下才会使用。

### 配置与个性化

| 命令 | 说明 |
|------|------|
| `/config` | 打开可视化配置面板，调整 Claude Code 的各项设置 |
| `/model` | 切换 AI 模型，可以在不同能力等级的模型之间选择（如 Opus/Sonnet/Haiku） |
| `/theme` | 更换界面主题，支持亮色/暗色等多种配色方案 |
| `/output-style` | 调整 Claude 的回复风格，可选择更简洁或更详细的输出方式 |
| `/permissions` | 管理工具权限规则，控制 Claude 可以执行哪些操作（如文件读写、命令执行） |
| `/hooks` | 配置事件钩子，在特定操作前后自动执行自定义脚本 |
| `/vim` | 为 Vim 用户启用 Vim 键位绑定，用熟悉的操作方式编辑文本 |
| `/terminal-setup` | 配置终端快捷键，如 Shift+Enter 换行，提升输入体验 |
| `/statusline` | 自定义状态栏显示内容，选择展示哪些信息（如模型、成本、上下文使用率） |
| `/sandbox` | 配置沙箱安全模式，限制 Claude 对系统的访问权限 |

### 扩展与集成

| 命令 | 说明 |
|------|------|
| `/mcp` | 管理 MCP（Model Context Protocol）服务器，连接外部工具和数据源 |
| `/plugin` | 安装和管理插件，扩展 Claude Code 的功能 |
| `/skills` | 查看可用的技能列表，技能是针对特定任务的预设能力 |
| `/agents` | 配置和管理自定义 Agent，为特定场景定制行为 |
| `/ide` | 管理 IDE 集成，查看和配置与编辑器的连接状态 |

### 统计与监控

| 命令 | 说明 |
|------|------|
| `/status` | 显示 Claude Code 完整状态，包括版本号、当前模型、登录账户、API 连接情况等 |
| `/stats` | 查看使用统计，了解你的 Claude Code 使用频率和模式 |
| `/usage` | 查看订阅计划的使用限额和剩余额度 |
| `/insights` | 生成详细的使用分析报告，了解你的工作模式和效率 |

## 🔧 特殊场景（按需使用）

这些命令在特定场景或一次性设置时使用。

### 账户管理

| 命令 | 说明 |
|------|------|
| `/login` | 使用 Anthropic 账户登录，首次使用或更换设备时需要 |
| `/logout` | 退出当前账户，清除登录状态 |
| `/upgrade` | 升级到 Max 订阅，获得更高的调用频率限制和更多 Opus 使用额度 |

### 平台集成

| 命令 | 说明 |
|------|------|
| `/chrome` | 配置 Chrome 浏览器扩展，让 Claude 可以在浏览器中辅助工作 |
| `/desktop` | 将当前会话无缝转移到 Claude Desktop 桌面应用继续 |
| `/mobile` | 显示二维码，扫码下载 Claude 手机 App |
| `/install-github-app` | 在 GitHub 仓库中安装 Claude GitHub App，实现 CI/CD 集成 |
| `/install-slack-app` | 将 Claude Slack 应用安装到你的 Slack 工作区 |

### 记忆与存储

| 命令 | 说明 |
|------|------|
| `/memory` | 编辑 Claude 的长期记忆文件，存储跨会话保留的重要信息（如项目约定、个人偏好） |

### 其他

| 命令 | 说明 |
|------|------|
| `/fast` | 切换快速模式（仅限 Opus 4.6），以更快的速度生成回复 |
| `/feedback` | 向 Anthropic 提交反馈，报告问题或建议新功能 |
| `/release-notes` | 查看最新版本的更新日志，了解新增功能和修复内容 |
| `/stickers` | 订购 Claude Code 官方贴纸（趣味功能） |
| `/exit` | 完全退出 Claude Code REPL 程序 |
